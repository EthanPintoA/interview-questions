# Number of Changes

Count the number of changes that occur in a given string. Note that a change does not include transitions between upper and lower case of the same letter.

## Example

```
Input: "ab"
Output: 1

Input: "aA"
Output: 0

Input: "aAa"
Output: 0

Input: "aBbc"
Output: 2

Input: "abCdE"
Output: 4
```
